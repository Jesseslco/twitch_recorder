# twitch_recorder
[![HitCount](http://hits.dwyl.io/Jesseslco/twitch_recorder.svg)](http://github.com/Jesseslco/twitch_recorder)

> python3.8 required

> ffmpeg required

> Support Win Linux

## Usage
`pip install -r requirements.txt`

`cd twitch_recorder/`

`python main.py`

press q or Q in terminal to stop
